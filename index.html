Works

MenuMaker

MenuMaker is a full-scale web application that utilizes Node.js, Express, and MongoDB in the backend and EJS, CSS, and JavaScript on the frontend. This app allows users to craft recipes and/or form menus with their own or community generated recipes. This is a great place to plan your meal prepping, share recipes with your friends, or just see what recipes the world has to offer!

FlyHigher!

FlyHigher is a desktop game that allows you to navigate a flying hot air balloon through falling spikes. The higher the altitude, the higher your score! This project uses 2D Unity, C#, and pixel art made within Photoshop. You can download it using the above icon if you are on a Windows Machine.

Constructor

Constructor is a two player desktop game that you can play on the same keyboard! Use A, Z, S, and Space to control the blue side and the arrow keys to control the red side. This game was my first ever attempt at a project so it is a bit rough around the edges. You can download it using the above icon if you are on a Windows Machine.
