My Works

MenuMaker

MenuMaker is a web application that utilizes Node.js, Express, and MongoDB in the backend and EJS, CSS, and JavaScript on the frontend. This app allows users to craft recipes and/or form menus with their own or community generated recipes. This is a great place to plan your meal prepping, browse and share recipes with the world, or generate ideas for your next home-cooked meal!

FlyHigher!

FlyHigher is a desktop game that allows you to navigate a flying hot air balloon with your mouse through falling spikes. The higher the altitude, the higher your score! This project uses Unity with C# scripting, and features pixel-art made within Photoshop. You can download it and play using the above icon if you are on a Windows Machine.

Thought-Pot

Thought-Pot is a small web application that allows the user to create positive reflections from templates and save them as a .PNG file! The site also contains links for every CSU's Counseling and Psychological Services. This application was made for Marinahacks 2021, a yearly 12-hour hackathon hosted by CSULB Women in computing (WiC). The project uses HTML5, CSS3, JavaScript, Bootstrap 4, and jQuery to produce a minimalistically-designed and responsive website.
